## Setup Instructions

1.  **Clone & Install**:

    ```bash
    git clone https://github.com/AshwinJ5/Nike-Ecommerce.git
    cd skill-test-nextjs
    npm install
    ```

2.  **Environment**:
    Create `.env.local` with necessary API keys.

3.  **Run**:

    ```bash
    npm run dev
    ```

4.  **Build**:
    ```bash
    npm run build
    npm start
    ```

## Tech Decisions

-   **Next.js 16 (App Router)**: For SSR, SEO, and modern routing.
-   **Tailwind CSS v4**: For rapid, utility-first styling.
-   **Zustand**: For simple, lightweight global state management.
-   **GSAP**: For complex, high-performance animations (Product Cards).
-   **React Hot Toast**: For accessible user notifications.
